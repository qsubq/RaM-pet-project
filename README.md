# RaM application
This app can showing Rick and Morty character

## Stack:
- Kotlin
- Picasso
- MVVM
- Room
- Retrofit2
- RecyclerView
- Kotlin Coroutines

## Description:
App has been created according to the Single Activity with navigation_component
